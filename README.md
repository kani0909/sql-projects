# SQL проекты

Коллекция учебных проектов по SQL (курс School 21 / Ecole 42).

## 📚 Содержание

| Название | Описание |
|----------|----------|
| **Day 02** | SQL операции, JOIN, агрегатные функции |
| **Day 03** | Подзапросы, вьюхи, транзакции |
| **Day 04** | Вложенные запросы, CTE |
| **Day 05** | Индексы, оптимизация запросов |
| **Team 00** | Командный проект (SQL обмен данными) |
| **Snapshots** | Работа со снимками данных |
| **Integrating features** | Интеграция различных SQL возможностей |
| **OLAP** | Аналитическая обработка данных |
| **Isolation levels** | Уровни изоляции транзакций |
| **Functions and procs** | Хранимые функции и процедуры |
| **Warehouse fundamentals** | Основы хранилищ данных |

## 🛠 Технологии

- **PL/pgSQL**
- PostgreSQL
- Транзакции и уровни изоляции
- OLAP операции

## 📁 Структура

Каждый проект содержит:
- `src/` — SQL скрипты с решениями
- `materials/` — вспомогательные материалы
- `misc/` — изображения и схемы
- `README.md` — описание заданий
- `model.sql` — схема базы данных

## 🚀 Запуск

Для выполнения скриптов необходима СУБД PostgreSQL:

```bash
# Подключиться к базе
psql -d your_database -U your_user

# Выполнить скрипт
\i src/ex00/dayXX_ex00.sql
```

## 📝 Примеры заданий

- Оптимизация запросов с использованием индексов
- Написание хранимых процедур и функций
- Работа с транзакциями и блокировками
- Аналитические запросы с оконными функциями
